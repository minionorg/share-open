add #

 add<T>(array: T[], item: T, index?: number): T[]

数组添加元素

参数 #

  • array — 将要添加的数组
  • item — 将要添加的元素
  • index — 添加的位置，可选参数

返回值 #

  • 返回新的数组

案例 #

add([], 'a') = ['a']
add([], 1) = [1]
add([1, '2'], 'a') = [1, '2', 'a']
add([1], { a: 1 }) = [1, { a: 1 }]
add([1], { a: 1 }, -1) = [{ a: 1 }, 1]
add([1, 2], 3, 2) = [1, 2, 3]
add([1, 2], 3, 1) = [1, 3, 2]
add(1, 1) = throw TypeError
add([], 1, 'a') = throw TypeError
